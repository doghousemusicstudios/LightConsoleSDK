Console Discovery

ShowUp detects consoles on the network and offers to set up coexistence

When ShowUp connects to a network, it runs ArtPoll discovery. If a known lighting console responds, this notification appears. Click the button below to simulate discovery.


How It Works

Phase 1

1. ShowUp sends ArtPoll broadcast packets on the local network
2. All Art-Net devices respond with ArtPollReply (including OEM code)
3. ShowUp matches OEM codes against its known console database
4. Detection includes universe scanning — ShowUp knows which universes the console uses
5. The notification gives the user a clear, one-tap path to coexistence setup
