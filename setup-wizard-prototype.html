Auto-Discovery
ShowUp scans your network for lighting consoles via ArtPoll broadcast.

How Discovery Works

ShowUp sends ArtPoll packets to 255.255.255.255:6454 and common subnet broadcasts. Consoles respond with ArtPollReply containing their OEM code, name, active universes, and IP address. ShowUp matches the OEM code against its built-in console database.

Discovery is automatic — it runs whenever ShowUp is connected to a network. No user action needed.
